Sistema operativo:
Es el software principal del computador o servidor. Administra los recursos y permite ejecutar aplicaciones.

Node.js:
Es un entorno de ejecución que permite ejecutar JavaScript fuera del navegador y crear servidores.

Aplicación server.js:
Es el archivo donde está el código del servidor. Define las rutas y las respuestas que recibirán los usuarios.

Puerto:
Es el punto de comunicación por donde el servidor recibe solicitudes. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros de eventos y actividades del servidor. Ayudan a monitorear el funcionamiento y detectar errores.
